# Проект "Processing" и "generators"
## Описание проекта
Добавление новых функциональностей для существующего проекта
в рамках домашней работы по курсу разработка на Python 
(продвинутый GIT).
Данный функционал предполагает: 
прием в качестве вводных данных списка словарей
и его дальнейшую сортировку по критериям (наличие ключа 'state : EXECUTED' и дата). 
V2.0 
Добавлен новый модуль в "generators".
В котором реализованы новые функции-генераторы
filter_by_currency - принимает на вход список словарей, представляющих транзакции.
Функция должна возвращать итератор, который поочередно выдает транзакции, 
где валюта операции соответствует заданной (например, USD).
transaction_descriptions - который принимает список словарей с транзакциями и 
возвращает описание каждой операции по очереди.
card_number_generator - который выдает номера банковских карт в формате 
XXXX XXXX XXXX XXXX.
Добавлен новый модуль decorators.
В котором реализована функция записи логов
#### homework 13.1
Для работоспособности кода была использована библиотека pandas, openpyxl.
Проект развивается, и источником данных о финансовых транзакциях теперь 
может быть не только JSON-файл, но и CSV- 
или XLSX-файл.
Добавлен новый модуль transaction_function.py.
В нем реализованы 2 функции: read_transactions_from_csv и read_transactions_from_excel
эти функции просты отвечают за распаковку данных в формате "список словарей"
из файлов формата .csx .xlsx. 
Также были написаны тесты к данным функциям.
Для демонстрации работы кода в transaction_function.py закомменчена функция пуска.
#### homework 13.2
Добавлены новые функциональности
Реализована часть логики, предполагающую работу с регулярными выражениями.
Используя для этого библиотеку re.
В модуле transaction_finder.py - реализованы функции подсчета 
количества транзакций по категориям, используя Counter и фильтрации.
В модуле main.py реализована основная функция и функции-хелперы, а также реализован
пользовательский интерфейс.
Далее в "Примерах" подробная информация о работе программы.
